HTML & JavaScript Notes

The <body> tag defines the main content of an HTML document which displays on the browser. It can contain text content, lists, paragraphs, headings, images, tables, links, videos, etc.

The <script> tag contains the script statements.

To select HTML elements, JS often uses document.getElementById().
